# DocketAlly Landing Page

Static landing page for [docketally.com](https://docketally.com) — workplace documentation platform.

## Deploy

Deployed on Vercel as a static site. Push to `main` to deploy.

## Stack

- Static HTML/CSS
- Fonts: Fraunces (serif), Manrope (sans), JetBrains Mono (mono) via Google Fonts
- No build step required

## Notes

- Photos are currently embedded as base64 data URIs. Swap to hosted images for production performance.
- Domain: docketally.com (configure in Vercel dashboard)
